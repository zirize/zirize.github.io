전체 종합 PDF

모든 라벨이 한 파일에 담긴 PDF입니다. 한 번에 인쇄하거나 필요한 페이지만 선택해서 출력하세요.

📥 recycle_all.pdf 다운로드

사용 방법

  1. 원하는 라벨의 PDF 파일을 다운로드합니다.
  2. PDF를 열어 A4 용지에 실제 크기(100%)로 인쇄합니다.
  3. 인쇄한 라벨을 해당 분리수거 통에 붙이거나 코팅하여 사용합니다.
  4. 크기 조정이나 색상 변경이 필요하면 SVG 파일을 Inkscape 등의 프로그램으로 편집하세요.
